Publicar Resultados

Los cambios que hacés (resultados, sorteo, playoffs) se guardan solo en este navegador. Para que los espectadores los vean en GitHub Pages, hacé lo siguiente:

  1. Tocá "Descargar data.json" abajo (se descarga a tu carpeta de Descargas).
  2. Reemplazá el archivo data.json dentro de la carpeta de tu repositorio por el que acabás de descargar.
  3. Ejecutá en la terminal, dentro de la carpeta del repo:
    git add data.json
    git commit -m "Actualizar resultados"
    git push
  4. En 1-2 minutos GitHub Pages actualiza el sitio con los nuevos datos.
